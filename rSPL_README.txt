--- Instructions to run rSPL program ---

1) The rSPL folder contains all the files required to construct the rSPL interpreter. They are all the sPL files (containing the modifications for rSPL). 

2) Also contained in the rSPL folder is a shell script file, bincomp7.sh . 

3) To obtain the rSPL interpreter, named spli, run the following command in terminal: 

             ./bincomp7.sh

4) Following this, the rSPL program files (r1.rspl - r12.rspl) may be run as follows:

             ./spli file_name
